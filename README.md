# Taj Store

Taj Store is a gold shop management system built using Java Swing and MySQL.

## Features
- Customer Management
- Supplier Management
- Sales and Purchases
- Inventory Tracking
- Reports and Dashboard

## Technologies
- Java
- Swing
- MySQL
- JDBC
- NetBeans

## Project Files
- Database scripts
- ER Diagram
- Normalization
- Source Code
- Presentation
